# DAY9 — 從「規則式視覺」進化到「深度學習分類」

DAY7 用 OpenCV 教電腦「看形狀」— 我們自己設計條件（面積、頂點數、圓度）去分類。今天翻頁：教電腦**自己學**怎麼看。用兩條路：

1. **Teachable Machine**（Google 的瀏覽器工具）—— 不寫程式碼就能訓練分類器
2. **CNN（卷積神經網路）** —— 打開 Teachable Machine 底下的引擎，親手寫一個

---

## 為什麼 DAY7 → DAY9 是自然銜接

| 面向 | DAY7 規則式 | DAY9 CNN |
|---|---|---|
| 誰決定「什麼是三角形」 | **你**（if vertices == 3）| **模型**（看很多樣本自己歸納）|
| 換一個問題（例如「這是不是貓」）| 幾乎沒辦法用規則寫 | 換資料就能重訓 |
| 訓練時間 | 0 秒 | 秒到數小時 |
| 對噪音 / 角度 / 光線 | 脆弱 | 資料夠多就強 |
| 何時用 | 邊界清楚、樣本少 | 邊界模糊、樣本多 |

**不是誰取代誰**，是工具箱多一件。

## 環境準備

```bash
pip install tensorflow opencv-python matplotlib numpy pillow
```

TensorFlow 在 Windows CPU 版本裝下去就能跑。有 GPU + CUDA 更快，但 04 的 MNIST CPU 訓練也只要幾分鐘。
